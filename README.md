Flask Twitter App

Flask App Task:

1. Create App in Python with proper requirement and installation using virtual environment
2. Database creation and table migration
3. Authentication and all those required API like Register, Login, Forgot Password, Reset Password and Logout
4. Microblogging site like Twitter with 15 words limitation for blogs
5. Allow image upload to users while blogging
6. Profile creation and edit option for logged in user
7. Show blogs to other users in LIFO format
8. Provide like counter when other users like blog and it can be reflected when page is refreshed
9. Proper error handling at each level
10. Please use Jinja template or simple HTML CSS. Design is not so important features should be working.
